# UTS
# Rekayasa Aplikasi Terdistribusi
Berikut adalah file coding chat app yang telah di buat oleh Ordrick dari Universitas Pelita Harapan Medan.

Pertama, untuk menjalankan aplikasi chat app ini yang harus dilakukan adalah dengan mendownload file <b>server.py</b> dan <b>client.py</b> terlebih dahulu. Jika sudah selesai di download maka boleh membuka atau menjalankan file server.py dan client.py di command prompt ataupun membuka filenya secara langsung. Untuk menjalankan aplikasi ini dibutuhkan python versi 3 keatas.

# **CARA RUN CHAT APP DI COMMAND PROMPT**
1. Download file server.py dan client.py
2. Buka dua(2) command prompt yang berbeda.
3. Direct(cd) ke folder dimana server.py dan client.py di simpan.
4. Setelah di direct ke folder yang tepat, yang pertama kita akan memerlukan server untuk membuka ruang chat jadi langsung saja ketik <b>server.py</b> dan enter.
5. Jika ruang chat sudah dibuat, di command prompt yang satunya lagi ketik <b>client.py</b> dan enter.
6. Masukkan server host yang telah di buat di server.py tadi.
7. Selamat koneksi suskes dan sudah dapat melakukan chatting satu sama lain.
